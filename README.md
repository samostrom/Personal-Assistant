# Navi Capture — Voice-to-Todoist Integration

## 🚀 Overview
A frictionless voice capture tool that lets you record thoughts quickly from your phone, pause naturally, and automatically store them in folders in Todoist. Built to explore low-friction MVP design and integration pipelines with AI transcription.

## 🎯 Goal
Capture and organize ideas instantly, without unlocking the phone or opening multiple apps. Solve real cognitive offloading pain.

## 🛠 Tech Stack
- React (PWA) for front-end capture
- Node.js + Express backend
- OpenAI Whisper API for transcription
- Todoist REST API for task creation
